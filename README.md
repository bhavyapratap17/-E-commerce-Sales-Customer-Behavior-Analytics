# 🛒 E-commerce Sales & Customer Behavior Analytics

## 📌 Project Overview
This project delivers a **Power BI dashboard** designed to analyze customer purchase behavior, seasonal sales patterns, and product profitability.  
It leverages **RFM segmentation** and **predictive KPIs** to identify high-value customers, churn risks, and optimize decision-making for e-commerce businesses.

---

## 🚀 Key Features
- Interactive **Power BI dashboard** for sales, customers, and products.  
- **RFM (Recency, Frequency, Monetary) segmentation** to classify customers.  
- Seasonal and product-level profitability analysis.  
- Predictive KPIs for identifying **high-value customers** and **churn risks**.  
- Drill-through insights for better decision-making and storytelling.
